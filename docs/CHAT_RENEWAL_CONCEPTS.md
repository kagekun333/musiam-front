# 伯爵チャット 全面リニューアル — コンセプトカタログ（選択用）

作成: 2026-06-27 / 目的: ①楽しませる ②親身に相談 ③毎月300万を売る最強の営業
方針: 複数案を多角的に提示 → 御社が選択 → 実装。奇抜案も含む。

> 大前提（倫理ガード）: 相手が本当に弱っている（distress）局面では**絶対に売らない**。
> 信頼を売上の手段にしすぎると一発で壊れる。「先に価値、後に提案」を鉄則にする。

## A. 人格・名前の体系（複数案）

### 案A-1：三伯爵・称号制（伯爵世界を磨く）
役割を3人に分ける。名前セットを3つ用意:
- セットα: **静寂の伯爵**(癒) / **星詠みの伯爵**(楽) / **差配の伯爵**(商)
- セットβ: **沈黙卿** / **夢幻卿** / **黒鍵卿**（“卿”で格を出す）
- セットγ: **夜伽の伯爵** / **幻燈の伯爵** / **宰相の伯爵**

### 案A-2：一人の伯爵＋秘書（★御社のイメージに最も近い）
- 語り手は**ただ一人の伯爵**（例: 「館の主」/ ABI伯爵）。相手を読み、癒やし・遊び・見極めを自在に切替。
- 商談だけを**秘書**が引き継ぐ。秘書名の候補: **秘書官ヴェイル** / **執事ノワール** / **司書セレネ**。
- 奇抜版: 秘書を**館の黒猫「ノワール」**にする（商談係が猫。SNS映え・親しみ・記憶に残る）。

### 案A-3：固有名の三人（親密さ）
- **ノクス**(夜・癒) / **オニロ**(夢・楽) / **セレス**(案内・商)。称号と併用。

### 案A-4：奇抜モチーフ
- **月の三相**: 新月伯（静）/ 満月伯（夢）/ 紅月伯（決断）。
- **仮面の三客**: 白仮面・銀仮面・金仮面（仮面舞踏会の夜会）。
- **館そのものが語る**: 人格ではなく“館の声”。部屋ごとに声色が変わる。
- **タロット擬人化**: 隠者（癒）・星（夢）・力（商）の大アルカナ。

## B. 会話アーキテクチャ（複数案）

### 案B-1：「一つの門・適応型」＋秘書ハンドオフ（★御社案）
- 入口は**1つだけ**。冒頭で軽い“読み”（観察質問1つ＝コールドリーディングの入口）で相手の状態を推定。
- 一人の伯爵が、相手に合わせて**癒やし→楽しませ→見極め**を自在に切替（扉を選ばせない＝迷路感ゼロ）。
- 信頼が育った頃合いで、最適な商材を**処方**のように差し出し、本格商談は**秘書**へ橋渡し。
- 長所: 関係が一本道で深まる/誘導が自然/実装もシンプル。短所: 1人格の表現力に全依存（→プロンプト設計が肝）。

### 案B-2：「三伯爵・リレー」
- 局面で伯爵が**交代登場**。癒やしが受け止め→夢見が楽しませ→執事が案内。バトンの演出が豪華。
- 長所: 各人格が立つ/見せ場がある。短所: 交代ロジックが複雑。

### 案B-3：「館の夜会・回遊」
- ユーザーは**客人**として夜会に招かれ、広間で複数の伯爵と歓談。回遊そのものが体験＝滞在時間↑。
- 長所: 没入・世界観最大。短所: 重い。

### 案B-4：「占い起点」
- まず短い占い/タロットで“今夜のあなた”を引き当て、その結果から伯爵が会話と商材へ。既存の占い資産と直結・バズ要素。

## C. セールス機構（信頼→販売の設計・複数案）

### C-1：秘書ハンドオフ型（推奨の核）
- 伯爵は**温めるだけ**。価格・見積・予約は**秘書**が別トーンで引き継ぐ（世界観を壊さず商談に入れる）。

### C-2：「今夜の処方箋」型
- 会話から相手に最適な1点を“処方”：落ち着きたい→睡眠BGMパック / 贈り物→オーダー曲 / 学びたい→講座 / 占い好き→占い×一曲。

### C-3：無料サンプル交換でリスト化
- 「壁紙1枚」「1曲」を**メール/LINEと引き換え**に。心理: 互恵性。再訪と販売の起点。

### C-4：コールドリーディング／説得設計（倫理ガード付き）
段階的に使う心理テクニック:
1. **観察→ラベリング**（“今夜は少し急いでいますね”）で信頼の入口。
2. **バーナム＋具体化**（誰にでも当たる一般化→相手の言葉で具体化）。
3. **ミラーリング**（語彙・速度を合わせる）。
4. **前提質問**（“どちらが近いですか”で選ばせ、決断を軽く）。
5. **互恵・希少**（先に価値→“今夜だけ”の限定感は控えめに）。
- ガード: distress時は販売停止／不安煽り禁止／誇大表現禁止。
