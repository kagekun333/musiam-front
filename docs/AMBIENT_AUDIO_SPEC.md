# 環境音ベッド 仕様（放送ハイブリッド R3 / オーナー生成用）

「国に常に音が流れている」を実現する**地方ごとの短いループ環境音**。あなたのAI制作で生成し、下記に置けば自動で鳴る設計にします。実曲とは別レイヤー（環境音＝土地の空気、実曲＝土地で出会う宝）。

## 形式
- ファイル形式: `.mp3`（128kbps以下で十分）＋ できれば `.ogg` も（互換用）。
- 長さ: **20〜45秒**、**シームレスにループ**できること（頭と尻が繋がる）。
- 音量: 控えめ・定常的（急な展開なし）。-18〜-14 LUFS目安。BGMとして邪魔しない“空気”。
- ステレオ可。容量は1本あたり〜1MB目標（合計数MB）。

## 置き場所（このパスに入れるだけで反映）
```
public/audio/realm/<地方id>.mp3
```
地方idは下記（atlas/regions.ts と一致）。**全部揃わなくてOK**、有る分だけ鳴らし、無い地方は“静寂”または隣の地方の音を流用します。

| 地方id | 地方名 | 音のイメージ（例） |
|---|---|---|
| shrine | 神々の社 | 荘厳なドローン、遠い鈴、空気感 |
| highland | 祝祭の高地 | 遠い祭囃子、太鼓の残響、風 |
| coast | 巡礼の海岸 | 波、潮騒、遠い民族打楽器 |
| reverie | 陶酔の樹海 | 反復するパッド、催眠的、虫の音 |
| dawn | 払暁の岸 | 静かなピアノ/パッド、鳥、朝の光 |
| alley | 追憶の路地 | 雨、遠い弦、もの寂しい |
| citadel | 緊迫の城砦 | 低い緊張音、金属、鼓動 |
| market | 陽だまりの市井 | 穏やかな生活音、軽い旋律 |
| library | 物語の書架 | 紙の音、時計、静寂のドローン |
| skyfield | 空撮の高地 | 風、高度、広がるシンセ |
| frontier | 未踏の辺境 | 未知の微音、まばらな点描 |

## 任意: 入領の効果音
- `public/audio/sfx/enter.mp3` … 「国に入る」クリック時の一瞬の音（封蝋が割れる/扉/鐘）。0.5〜2秒。

## 仕組み（私が実装）
- 入領クリックで音声解錠（ブラウザの自動再生制限対策）。
- 地方を移動すると `gain` で**クロスフェード**（数秒）。
- ミュート/音量つまみ、初回は小音量。reduced-motion/省データ時はオフ可能。
- 実曲は「今、領内に流れている一曲」枠で表示→クリックで配信/Spotify埋め込み。

> まずは2〜3本（例: shrine / alley / coast）だけ作って置いてくれれば、私が鳴る状態にして体感を確認できます。残りは後追いで差し替え可。
