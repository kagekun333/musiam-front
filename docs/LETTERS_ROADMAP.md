## ワークストリーム1: 今後の新規手紙の構成多様化

### 現状分析
`content/letters/*.md` 552通のうち546通（99%）が、2025-01-01の初回から一貫して同一の4部構成・ほぼ全通が正確に25行という、非常に硬直したテンプレートで書かれている。

```
[一言书き出し]
## 館より
[その日の出来事・話題]
## 伯爵の独白
[振り返り・独白。/works等への軽い導線]
## よくある質問
**Q. ...** / [自問自答1問]
**ABI伯爵**
```

話題（対戦相手、経済指標、その日のニュース）は日々変わるが、器が552回とも同じなので、続けて読むと似た印象を受ける。過去分はこのまま残し（年代記としての記録性を優先）、**今後の新規分から構成を変えていく**方針で合意済み。

このリポジトリには手紙を自動生成するスクリプトが見当たらない（外部のAI対話等で書かれた本文がMarkdownとしてそのままコミットされる運用と推測）。したがって「多様化」は自動化ではなく、**執筆時に参照するガイド**として機能させる。

### 提案テンプレート（ローテーション想定・確定ではない）

| # | 型 | 構成 | 使いどころ |
|---|---|---|---|
| A | 現行の定型 | 一言→館より→独白→FAQ1問→署名 | 従来通りでよい日。使用頻度を下げ、目安 週1〜2回程度に |
| B | 物語型 | 見出しなしの一続きの散文。長さも可変（10〜40行） | 静かな日、内省的な話題 |
| C | 制作ログ型 | その日にできた作品1つを主役にし、着想〜選別〜仕上げの一場面を具体的に描く。FAQなし | 新曲・新刊リリース日 |
| D | 対話再現型 | 伯爵とAIの短いやり取りを再構成し、そこから一段落の気づきで締める | 制作上の判断・葛藤があった日 |
| E | 短文型 | 見出しなし5〜10行のみ。儀式的な「今日も書きました」に留める | 特筆すべきことがない日・多忙な日 |

補足の方針案:
- 「よくある質問」の自問自答は毎回ではなく、5通に1通程度に間引く（現状538/552で使用=ほぼ毎回）。
- 長さを25行固定から意図的にばらつかせる。
- 「**ABI伯爵**」の署名は一貫性の演出として毎回維持してよい。
- 過去の手紙への軽いリンク（例:「あの日と同じ話を今日もしている」）を時々挟むと、単発でも過去回とのつながりが生まれる（今回追加した前後ナビ・年月索引と相性が良い）。

### 進め方（案）
1. 上記テンプレートの型・使用比率をあなたが確定（このドキュメントへのフィードバックでも可）。
2. 次回以降の執筆（AIとの対話セッション等）でこの表を渡し、その日の内容に合う型を選んでもらう。
3. 数週間試して読み味を確認、必要なら型を増減。

## ワークストリーム2: アフィリエイトリンクの導入（今後分＋過去分）

### 実装済みの技術基盤（コード対応済み・公開待ちではなく既に有効）
`src/lib/letters.ts` の `renderMarkdown()` に専用記法を追加した。

```markdown
通常のリンクは [これまで通り](/works) のまま。
アフィリエイト/PRリンクは末尾に {sponsored} を付けるだけ:
愛用している道具は [Suno Pro](https://example.com/aff-link){sponsored} です。
```

`{sponsored}` を付けると自動的に:
- 目に見える **PR** バッジが表示される
- `rel="sponsored nofollow noopener"` が付与される（ステマ規制対応・検索エンジンへの明示）
- `target="_blank"` で新規タブに開く

書き手が意識するのはこの記法だけでよく、開示表記を毎回手打ちしたり忘れたりするリスクがない。

### 法令メモ（法的助言ではなく一般的な制度情報）
2023年10月施行のステマ規制（景品表示法5条3号）により、報酬が発生するリンクは広告だと消費者に分かる表示が必要（[消費者庁](https://www.caa.go.jp/policies/policy/representation/fair_labeling/stealth_marketing)）。上記の「PR」バッジ表示はこの要請を満たす狙いで設計している。実際の運用（表記文言など）に不安があれば、最終的には専門家に確認するのが望ましい。

### 過去546通の見直しについて — 分かったこと
「本文中の商品名をキーワード検索して機械的に洗い出す」方式を試したところ、精度が低いことが分かった。例えば `2025-05-23-coffee-love-and-the-mirror.md` は「コーヒー」で検索に引っかかるが、実際は自作楽曲『Coffee Love』の話で、飲料のコーヒーとは無関係。手紙は基本的に時事ニュースと自作品の話が中心で、市販品への具体的な言及自体が少ないため、**「既存の言及を拾って機械的にリンク化する」よりも、「テーマ別に狙って新規の一言＋リンクを足す」ほうが現実的**。

候補になりそうなテーマ（既存の手紙群から）:
- AI制作ツール・ワークフローに触れる回（`prompt-craft.md`, `human-role-in-ai-music.md` など）→ 使用しているAIツール・書籍等
- 工房講座・法人依頼に触れる回 → 自社導線なのでアフィリエイトではなく通常の内部リンクで十分
- ガジェット・エンタメの実ニュースに触れた日（Switch 2発売回など）→ 該当製品
- 「収益の実際」等の運営を語る回（`the-real-numbers.md`）→ 実際に使っている配信・制作ツール

### 必要な情報（あなたから）
実際にリンクを入れるには、以下が必要:
1. 使うASP/プログラム（Amazonアソシエイト、楽天アフィリエイト、A8.netなど）とアカウントの有無
2. 優先したいジャンル・商品（音楽制作ツール系／書籍系／ガジェット系など、方向性だけでも）
3. 実際のアフィリエイトリンク（URL）— これは私が代わりに取得・生成することはできないため、都度共有いただく形になる
4. 「PR」の表記文言でよいか（「広告」「プロモーション」等の代替案があれば）

### 進め方（案）
1. まず3〜5通のパイロットで試す（新規に書く手紙、または過去の中から相性の良い1〜2通）。
2. 見え方・トーンを確認。
3. 問題なければ、過去分は「狙ったテーマの回に一言＋リンクを追記する」形で件数を広げる。全546通を一律に見直すのではなく、相性の良い回だけを対象にする想定。
